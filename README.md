## Prerequisites

You will need the following things properly installed on your computer.

* [Git](https://git-scm.com/)
* [Node.js](https://nodejs.org/) (with npm)
* [Ember CLI](https://ember-cli.com/)
* [Google Chrome](https://google.com/chrome/)

## Installation

* `git clone <repository-url>` this repository
* `cd sgtoolkit`
* `npm install`

## Running / Development

* `ember serve`
* Visit your app at [http://localhost:4200](http://localhost:4200).
* Visit your tests at [http://localhost:4200/tests](http://localhost:4200/tests).

## Deploying docs

* git checkout master
* ember github-pages:commit --message "update docs"
* manually update asset links in index.html to add github pages top level repo name

For Example

  /assets/sgtoolkit-18bc90cd83f34e7b507bd895e4713df1.css

should be updated to

  /Design-Template/assets/sgtoolkit-18bc90cd83f34e7b507bd895e4713df1.css
